# Mi Primer Proyecto - Ingeniería en Sistemas

Este es mi primer repositorio gestionado con **Git** y subido a **GitHub**. Aquí estaré guardando mis prácticas de programación y ejercicios de clase.

## Contenido
Actualmente, el repositorio contiene:
* **HolaMundo.java**: Un programa básico en Java realizado en VS Code.
* **Configuración inicial**: Estructura base del repositorio.

## Tecnologías utilizadas
* **Lenguaje:** Java
* **Editor:** Visual Studio Code
* **Control de Versiones:** Git
* **Plataforma:** GitHub

## Cómo ejecutar el proyecto
Para correr el código de Java en tu máquina local:
1. Clona el repositorio.
2. Abre la terminal.
3. Compila con: `javac HolaMundo.java`
4. Ejecuta con: `java HolaMundo`
